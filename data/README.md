# Data Download Instructions

This document provides guidelines on how to download datasets from various sources for our drought prediction project.

## 1. GLEAM
- **Source:** [GLEAM](http://www.gleam.eu/)
- **Download Instructions:** Visit the GLEAM website and navigate to the data section. Follow their instructions to download the relevant data products.

## 2. NASA POWER MERRA-2
- **Source:** [NASA POWER](https://power.larc.nasa.gov/)
- **Download Instructions:** Use the NASA POWER website to access the MERRA-2 data. Select the parameters, time range, and geographic area before downloading.

## 3. SoilGrids
- **Source:** [SoilGrids](https://soilgrids.org/)
- **Download Instructions:** SoilGrids offers global soil property maps. You can download data via their API or user interface.

## 4. SRTM
- **Source:** [SRTM](https://www2.jpl.nasa.gov/srtm/)
- **Download Instructions:** Access the NASA SRTM website to download the digital elevation models. Choose the desired tiles based on your area of interest.

## 5. ESA WorldCover
- **Source:** [ESA WorldCover](https://worldcover2020.org/)
- **Download Instructions:** Visit the ESA WorldCover website to download global land cover data for 2020.

## 6. WorldPop
- **Source:** [WorldPop](https://www.worldpop.org/)
- **Download Instructions:** Navigate to the WorldPop website to acquire population datasets by selecting the desired parameters and regions.

## 7. GADM Shapefiles
- **Source:** [GADM](https://gadm.org/)
- **Download Instructions:** Visit the GADM website to download administrative level shapefiles for various countries. Select the country for which you need the shapefiles.

Ensure to follow any licensing agreements associated with each dataset.

# Repository Structure

This repository contains various components related to drought prediction in Tunisia. Below is an overview of its structure:

- **data/**: Contains datasets used for training and testing the model.
- **notebooks/**: Jupyter notebooks for exploratory data analysis and visualizations.
- **src/**: Source code for model training, evaluation, and predictions.
- **models/**: Pre-trained models and model artifacts.
- **requirements.txt**: Contains all the Python package dependencies used in this project.
- **README.md**: General information about the project, including setup and usage instructions.

## Usage
Make sure to follow the setup instructions in `README.md` to run the project effectively.
